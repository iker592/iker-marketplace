# Second Brain

A personal knowledge management system for capturing and retrieving information.

## Capabilities

- **Capture**: Save notes, ideas, code snippets, and references
- **Organize**: Tag and categorize knowledge for easy retrieval
- **Retrieve**: Search and recall information when needed
- **Connect**: Link related concepts and ideas

## Usage

Use `/second-brain` to:
- Add new knowledge entries
- Search existing entries
- List recent captures
- Connect related ideas

## Storage

Knowledge is stored in `~/.second-brain/` with the following structure:
```
~/.second-brain/
├── notes/
├── references/
├── snippets/
└── index.json
```
